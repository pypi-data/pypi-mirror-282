# Traderman
Python SDK to build, train, stress-test, deploy and monitor trading strategies

# installation

## Using pip

```shell
pip install traderman

```

## Clone repo

```shell
git clone
```

# Usage

1. install with pip
2. Add `api_key` and `secret_key` as environment variables
3. run `python3 -m examples/basic_trading.py`
